# 🐍🎮 Game Ular Python – Arcade Seru by Firla

Game ular klasik dengan nuansa retro, dibuat menggunakan **Python** dan **Pygame**.  
Ringan, seru, dan bikin nostalgia – lengkap dengan musik latar dan efek suara! ✨
