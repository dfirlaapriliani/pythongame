🐍 Pixel Retro Snake Game by Firla

🎮 Cara Bermain:
- Gerakkan ular pakai tombol arah (↑ ↓ ← →)
- Makan buah pixel lucu 🍒 untuk nambah skor dan panjang
- Jangan nabrak dinding atau badan sendiri ya!

🎵 Musik:
- Suara 8-bit retro saat makan
- Musik chiptune lucu dan santai sebagai latar

▶ Jalankan:
1. Install pygame:
   pip install pygame
2. Jalankan:
   python main.py

Enjoy Firla's Happy Retro Game! 🎉🎮
